Memory Game

Игра на память на Qt. Классическая игра "Найди пару".

Описание

Игрок открывает карточки и ищет одинаковые пары. Игра поддерживает поля от 2×2 до 8×8, ведёт статистику ходов и времени, сохраняет рекорды и показывает 10 лучших результатов для каждого размера поля.

Функционал

  • Выбор размера поля (2×2, 4×4, 6×6, 8×8)
  • Открытие карточек и поиск пар
  • Подсчёт ходов и времени
  • Теоретический минимум и максимум ходов
  • Сохранение истории игр в файл records.txt
  • Топ-10 лучших результатов с группировкой по размеру поля
  • Обработка ошибок (исключения)
  • Модульные тесты (Google Test)

Установка и сборка

Требования

  • CMake 3.16+
  • Qt 6 (или Qt 5)
  • Компилятор с поддержкой C++17
