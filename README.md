# Memory Game

Игра на память с карточками, написанная на C++ с использованием Qt. Игрок должен находить пары одинаковых изображений.

## Основные возможности

- **Геймплей**: Поиск пар карточек на поле от 2×2 до 8×8
- **Уровни сложности**: Выбор размера поля (2×2, 4×4, 6×6, 8×8)
- **Статистика**: Подсчёт ходов, времени, теоретический минимум и максимум
- **Сохранение прогресса**: Автоматическое сохранение истории игр в файл `records.txt`
- **Топ-10 лучших результатов**: Просмотр 10 лучших попыток для каждого размера поля

## Правила игры

- Выберите размер поля (2×2, 4×4, 6×6 или 8×8)
- Кликните по карточке, чтобы перевернуть её
- Найдите две одинаковые карточки подряд
- При совпадении карточки блокируются
- Игра продолжается, пока не найдены все пары
- В конце игры показывается статистика: ходы, время, теоретический минимум и максимум

## Технические требования

### Необходимые ресурсы:

- Папка `images/` с картинками (1.jpg, 2.jpg, ...)
- Для поля 8×8 требуется 32 картинки

### Поддерживаемые форматы:

- Изображения: JPG
